Weding Planner

This is a wedding planner site that helps the user plan the seating for a wedding or other events. I add different colors to the site and added wedding related images to make the site's purposes more obvious and make the page look better.

After the user click on the button, the site will ask the user to first enter the number of people attneding the wedding.

Then, the site will ask for the number of tables at the wedding.

After the user enters both the number of people attending the wedding and the number of tables at the wedding, the site will organized the seats for the weddng as even as possible to ensure the guests are evenly distributed at the wedding, and it will display the number of tables with certain number of people in details to the user.
